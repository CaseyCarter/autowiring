Detailed Description

template<class object_type, class lock_type = std::mutex>
class atomic_object< object_type, lock_type >

summary> This class provides a standard thread-safe wrapper for data structures. /summary> remarks> This class supports two modes of use:

  • Atomic assignments which can be used to create a mutable local copy, or to modify the contents of atomic_object.
  • Held locks, via child classes that manipulate the protected lock member. /remarks>
