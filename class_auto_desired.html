Detailed Description

template<class T>
class AutoDesired< T >

Convenience class which attempts to inject type T and discards any exceptions

Use of this type is functionally equivalent to the following:

try { AutoCurrentContext()->Inject<T>(); catch(...) { AutoCurrentContext()->FilterException(); } Autowired<T> foo;

Users who wish to know whether an exception was thrown may replace uses of AutoDesired with the above and perform their own handling.
