A delayed instantiation fast pointer cast

This method will perform a correct fast pointer cast to the destination type without any knowledge of either T or U being required. It's able to do this as long as there is another location somewhere in the application where a fast_pointer_cast is being used, because the fast_pointer_cast type has a small registry of cast functions.
