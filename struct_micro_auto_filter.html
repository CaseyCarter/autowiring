Detailed Description

template<class Ret, class... Args>
struct MicroAutoFilter< Ret, Args >

Transmutes a function returning void to an instance that can be called by AutoPacket. The specializations distinguish between void and Deferred return types.

The default constructor yields an AutoFilter that does nothing.
