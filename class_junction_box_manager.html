This method checks whether eventoutputstream listeners for the given type still exist. For a given type in a hash, returns a vector of weak ptrs. Goes through the weak ptrs, locks them, erases dead ones. If any live ones found return true. Otherwise false. NOTE: this func does lazy cleanup on weakptrs ptng to suff that has fallen out of scope.
