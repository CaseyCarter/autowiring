Detailed Description

template<class T, class... TKey>
class AutoConfig< T, TKey >

Register an attribute with the AutoConfig system. For example

AutoConfig<int, struct MyNamespace, struct MyKey> m_myVal; defines the key "MyNamespace.MyKey"

The Namespace field is optional, so AutoConfig<int, struct MyKey> m_myVal; defines the key "MyKey"

AutoConfig values can be set from the AutoConfigManager. The string key is used as the identifier for the value
