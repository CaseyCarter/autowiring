Detailed Description

template<class T>
class Autowired< T >

An autowired template class that forms the foundation of the context consumer system

Parameters
TThe class whose type is to be found. Must be an EXACT match.

The autowired class offers a quick way to import or create an instance of a specified class in the local context.

This class may be safely used even when the member in question is an abstract type.

Allows a lambda function to be called when this slot is autowired

In contrast with CoreContext::NotifyWhenAutowired, the specified lambda will only be called as long as this Autowired slot has not been destroyed. If this slot is destroyed beforehand, the lambda will never be invoked.
