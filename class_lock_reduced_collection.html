Detailed Description

template<class T, class Hash>
class LockReducedCollection< T, Hash >

Implements a lock-free collection

A reduced-lock collection is a collection where there are no locks used to restrict read access, and write access is performed with optimistic locking. The lock reduced collection has very inefficient insertion, but very efficient search.

This collection is implemented with spin locks to make it more lightweight. If the collection becomes highly contended or is frequently edited, changes will be necessary.
