Detailed Description

template<class T>
class AutoRequired< T >

Similar to Autowired, Creates a new instance if this instance isn't autowired

This class is simply a convenience class and provides a declarative way to name a required dependency.

If type T has a static member function called New, the helper's Create routine will attempt call this function instead of the default constructor, even if the default constructor has been supplied, and even if the arity of the New routine is not zero.

To prevent this behavior, use a name other than New.
