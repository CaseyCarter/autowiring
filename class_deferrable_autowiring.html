std::weak_ptr<CoreContext> DeferrableAutowiring::m_context
protected

This is the context that was available at the time the autowiring was performed.

A weak reference is held in hroder to ensure proper teardown, otherwise the first created member of the context would generate a cyclic reference between this pointer and the context membership set.
