Detailed Description

template<class T, bool auto_ready = true>
class auto_out< T, auto_ready >

A utility type to positively declare an output to an AutoFilter

Parameters
auto_readySet if the output should be marked "ready" by default

The auto_output wrapper is a convenience alternative to accepting an AutoPacket and then requesting a checkout. It provides the additional benefit that the caller may declaratively specify whether the auto_output should be satisfied by default when the function returns, or if it must be manually satisfied with a direct call to Ready before the object falls out of scope
