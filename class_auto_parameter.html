Detailed Description

template<class T, class TKey>
class AutoParameter< T, TKey >

Register an AutoParameter with "AutoParam" namespace in AutoConfigListing. In addition to being the lookup string, the Key also implements:

  • static constexpr T Default()
  • (optional) static bool Validate(const T&)

AutoParameter uses AutoConfig under the hood and will use "AutoParam" as its namespace
