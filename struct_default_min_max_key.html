template<typename T, T DEFAULT, T MIN, T MAX>
struct DefaultMinMaxKey< T, DEFAULT, MIN, MAX >

Base class for providing an easy way to default, min and max values

Because the class names are used as keys to be stored in the AutoConfigManager, these classes cannot be used directly and must be subclassed:

struct MyDefaultMinMaxKey : DefaultMinMaxKey<int, 15, 10, 20> {}; AutoParameter<int, MyDefaultMinMaxKey> param;
