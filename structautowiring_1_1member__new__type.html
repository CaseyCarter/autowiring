Detailed Description

template<class T, factorytype ft = member_new_type_helper<T>::value>
struct autowiring::member_new_type< T, ft >

Detects whether the specified type T has a nonstatic factory New method

An auto-initializer routine is called after a shared pointer assignment has taken place, but before an entity is injected into the context. This affords a context member an opportunity to obtain and, optionally , distribute its own shared pointer before that entity becomes discoverable in that context.

Note that this call is actually made before the type is actually injected in the context, so an attempt to perform Autowired<T> from AutoInit will necessarily fail.
