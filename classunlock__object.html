Detailed Description

template<class object_type, class lock_type = std::mutex>
class unlock_object< object_type, lock_type >

summary> A reference to a atomic_object that guarantees both existence and exclusive mutability. /summary> remarks> An unlock_object cannot be copied by construction or assignment since it maintains access. An unlock_object cannot be used to extend the existence of an atomic_object beyond the span of existence of the unlock_object. /remarks>
